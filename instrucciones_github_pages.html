Pasos para publicar en GitHub Pages
1
Descarga el archivo index.html
Usa el botón de descarga que aparece arriba en este chat para guardar el archivo en tu computadora.
2
Crea un nuevo repositorio en GitHub
Ve a github.com → botón verde "New" → nómbralo exactamente:
carreraspormexico
3
Sube el archivo index.html
Dentro del repositorio → "Add file" → "Upload files" → arrastra el index.html → "Commit changes".
4
Activa GitHub Pages
Settings → Pages → Branch: main → carpeta: / (root) → Save.
En 1–2 minutos tu página estará en línea en:
https://tuusuario.github.io/carreraspormexico
5
Comparte la URL y prueba en el navegador
Abre la URL en Chrome, Safari o cualquier navegador. Los filtros de distancia y ciudad ya funcionan con JavaScript sin necesitar servidor.
